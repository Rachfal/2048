# 2048

My version of popular 2048 mobile game.

## Features:
- Custom game-boards (up to 16 x 16)
- Best scores and session stored in local storage
- PWA
- offline with Service Worker


## Technologies:
- JS (vanilla)
- SASS
- HTML
